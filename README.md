# Analyse Numérique avec Python (ANP) 🧮🐍

Bienvenue dans le dépôt **ANP**. Ce répertoire regroupe un ensemble de ressources, de Travaux Pratiques (TP) et de sujets d'examens (partiels) portant sur l'analyse numérique, implémentés en Python via des notebooks Jupyter. 

## 📖 Description

Ce dépôt a été pensé pour s'entraîner et réviser des concepts clés de l'analyse numérique et des mathématiques appliquées. Il contient à la fois des sujets vierges pour s'exercer et des corrections détaillées.

Les thématiques principales abordées dans ces notebooks incluent notamment :
* **L'intégration numérique** (formules élémentaires et composites : trapèzes, Simpson, etc.)
* **L'interpolation polynomiale** (Polynômes de Lagrange, Newton, etc.)
* **Diverses méthodes numériques** réparties au travers des différents TP.
